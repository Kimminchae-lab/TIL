1. 배상 비용 최소화

문제 링크: https://school.programmers.co.kr/courses/10916/lessons/69530

#### 내 답
```python
def solution(no, works):
    while no > 0:
        works = sorted(works)
        if works[-1] <= 0: 
            break
        works[-1] -= 1
        no -= 1
    return sum([i ** 2 for i in works])
```
- 내림차순으로 정렬했다가 works[-1]로 접근하는 방식으로 바꿈.
